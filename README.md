# BikeRoute (仮称)

> バイクの車種に応じた通行制限を考慮したGoogleマップ連携アプリ

## これは何?

50ccや125ccのバイクライダーが、自分のバイクで通れる道路だけを使ったルート案内をGoogleマップ経由で受け取れるWebアプリです。

Googleマップは「自動車専用道路を回避する」設定がないため、原付ライダーが知らずに進入禁止の道路に誘導される問題があります。このアプリは、出発地・目的地と車種を入力するだけで、車種に合わせた回避設定付きのGoogleマップを開きます。

## 対象ユーザー

- 50cc(原付一種・新基準原付): 自動車専用道路NG、高速道路NG
- 125cc(原付二種): 自動車専用道路NG、高速道路NG
- 126cc以上(普通二輪以上): 全道路OK

主な利用シーンは「ツーリング前の下調べ」です。

## なぜ作るのか

商用アプリ(ツーリングサポーター by NAVITIME など)はすでに存在しますが、月額有料です。一方、無料のGoogleマップは車種別の通行制限を考慮しません。その間を埋める「無料・OSS・コミュニティ駆動」のアプローチで、ライダー全般がアクセスできる選択肢を作ります。

## データソース

このプロジェクトは以下のデータを利用予定です。各ソースの利用可否は順次確認中。

- 日本二輪車普及安全協会(JMPSA)の通行規制区間情報(全国約500箇所)
- 国土交通省 国土数値情報の道路データ
- OpenStreetMap
- ユーザー投稿(写真付き、承認制)

## 免責事項

このアプリが提供する情報は参考情報です。**走行時は必ず実際の道路標識を確認し、最新の交通規則に従ってください。** アプリの案内に起因する違反・事故等について、本プロジェクトは一切の責任を負いません。
